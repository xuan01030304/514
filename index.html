HOW TO FIND VALUES FROM CSFLOAT
▸ price → rec["price"] ÷ 100 = USD Raw price in cents; divide by 100 for USD display 原始價格以分為單位，÷100 轉換為美元顯示
▸ float_value → item["float_value"] Continuous wear value 0–1; primary pricing driver in model 連續磨損值 0–1，為模型主要定價驅動因子
▸ paint_seed → item["paint_seed"] Pattern index 0–999; drives premium for top-100 collector seeds 花紋索引 0–999，Top 100 稀有 Seed 享有顯著溢價
▸ phase → item["phase"] (Doppler only) Doppler knife phase (P1–P4, Ruby, Sapphire, Emerald, B/T); adds categorical premium Doppler 刀相位（P1–P4、紅寶石、藍寶石、祖母綠等），需分類加成
▸ rarity_name → item["rarity_name"] Rarity tier string (e.g. "Covert"); controls band-width multiplier in conformal model 稀有度等級字串（如 "Covert"），控制符合預測區間的寬度倍數
▸ is_stattrak → item["is_stattrak"] Boolean; StatTrak™ applies ≈1.55× price multiplier (historically stable) 布林值；StatTrak™ 約有 1.55× 歷史穩定溢價倍數
▸ S₀/vol/mom → time_features[skin] Cell 4 From Notebook Cell 4: base price S₀ (30d VWAP), realized volatility, 30d momentum 來自 Notebook Cell 4：基準價 S₀（30天量加權均價）、已實現波動率、30天動量
QUICK CODE

seed = rec['item']['paint_seed']

▲ LIMITATIONS & REFLECTIONS
◈小樣本不穩定性。CSFloat ≤100 筆掛單。N<30 時 IPCA 退回 Observable Factor PCA（SVD），低流動性皮膚信心區間較寬。
◈無空頭機制。Steam 不支援做空。所有訊號均為 Long-Only，PASS = 暫不買，非賣出訊號。Kelly 為倉位大小，非方向訊號。
◈貼紙估值近似。歷史貼紙使用靜態查表。實際成交價因位置、磨損和需求差異顯著。
◈Major 賽季洩漏風險。Walk-forward 以 Major 邊界防止貼紙投機洩漏，但賽前 2-4 週仍進訓練集。
◈平台手續費不對稱。v11 Kelly 模型已納入買方費（Skinport 12%）與賣方費雙邊計算。b_net = (淨賣出 - 實付成本) / 實付成本，反映真實報酬率。
◈Meta-Confidence 冷啟動。需 ≥10 筆 OOS 觀測才啟動 Ridge 元學習器，此前退回啟發式規則。
◈v9 例外表為手動維護。StatTrak 衝突與 Float 方向表需定期更新。
